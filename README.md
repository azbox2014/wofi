# Wofi
Wofi is a launcher/menu program for wlroots based wayland compositors such as sway
## Dependencies
	libwayland-dev
	libgtk-3-dev
	pkg-config
## Building
	hg clone https://hg.sr.ht/~scoopta/wofi
	cd wofi/Release
	make
## Installing
	sudo cp wofi /usr/bin
## Uninstalling
	sudo rm /usr/bin/wofi
